# Task Manager API (Learning Project)

This is a small Node.js + Express project I built while learning :)

## Features
- Add new tasks
- Display all tasks
- Edit a task
- Delete a task

## How to Run
1. Clone this repo
2. Run `npm install`
3. Run `node index.js`
4. Test with Postman

## Note
This is only a practice project to understand the basics of APIs.
